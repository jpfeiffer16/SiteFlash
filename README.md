# SiteFlash
A tool for cloning sites locally

## Installing
`git clone git@github.com:jpfeiffer16/SiteFlash.git`

`cd SiteFlash`

`npm install`

## Usage

### Help listing
`node app.js --help`

### Clone a site
`node app.js -s test http://site-url`

### Export a site
`node app.js -s test -e`

### Export all sites
`node app.js -s test -e`
